Threat Hunting, Incident Response & Purple Teaming

This repository showcases a comprehensive, full-spectrum approach to enterprise security operations. It bridges the gap between reactive Incident Response (triaging and containing automated SIEM/XDR alerts) and proactive Purple Teaming (safely emulating adversary behaviors and conducting hypothesis-driven threat hunts).

The objective of this project is to sift through raw telemetry to track intrusions, map discovered TTPs to the MITRE ATT&CK Framework, execute active containment strategies, and engineer custom detection rules to continuously harden the environment against future attacks.

Tools & Technologies
Defensive Platforms (Blue): Microsoft Sentinel, Microsoft Defender for Endpoint (MDE)

Adversary Emulation (Red): Atomic Red Team, PowerShell (Living off the Land), Python, Bash.

Query Language: Kusto Query Language (KQL)

Frameworks: MITRE ATT&CK, NIST SP 800-61

Techniques: Hypothesis-Driven Hunting, Incident Containment, Telemetry Correlation, XDR Detection Engineering
